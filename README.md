# Doctolib BBL 07/11

## Quick Start

```bash
npm i
npm start
```

### `npm start` or `yarn start`

Runs the project in development mode.  
You can view your application at `http://localhost:3000`

The page will reload if you make edits.

### `npm t` or `yarn test`

Runs the test watcher (Jest) in an interactive mode.
